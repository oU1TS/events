# Implementation Plan: Raid Card Linking and Highlighting

This plan outlines the changes to support direct linking to specific raid cards in the UITS Event Raiders webpage. When a user accesses the site with a raid number in the URL (e.g., `#raid-1`), the website will automatically route to the "Raid Campaigns" section, scroll the card into view, and briefly highlight it. A "copy link" button will also be added to each card.

## User Review Required

No major architectural changes or breaking modifications. The hash routing will be extended to support `#raid-<number>`.

## Proposed Changes

---

### Component: Data Layer

#### [MODIFY] [raids.json](Documents/GitHub/[oU1TS]/events/raids.json)
Add the `"Raid_Num"` variable to the top of each raid card entry.

```json
[
  {
    "Raid_Num": 1,
    "title": "International AI Builders Congress 2026",
    ...
  },
  ...
]
```

---

### Component: Frontend / Styling

#### [MODIFY] [style.css](Documents/GitHub/[oU1TS]/events/style.css)
Add styles for the copy link button and the highlight animation.

- Create keyframe animation `highlightPulse` for a glowing box-shadow and accent border.
- Create class `.raid-card.highlighted` to apply the animation.
- Style `.copy-raid-link-btn` and container `.raid-meta-right` for alignment and hover effects.

---

### Component: Logic

#### [MODIFY] [app.js](Documents/GitHub/[oU1TS]/events/app.js)
Update routing, card rendering, and scrolling logic.

1. **SPA Routing**: Update `routePage` to match `#raid-<num>` hashes, internally redirect to `#past-raids` to display the section, but keep the hash intact.
2. **Card Rendering**: Add `raid-card-${raid.Raid_Num}` as the ID to the raid card element.
3. **Copy Button**: Render the copy button in the card meta header and bind a click listener to copy the unique link with feedback.
4. **Scroll & Highlight**: Implement `scrollToRaid(raidNum)` to center the card on screen and trigger the highlight pulse.

## Verification Plan

### Manual Verification
1. Open the page locally and check that each raid card has a copy icon button.
2. Click the copy icon and verify that the icon changes to a checkmark, and the correct URL (e.g. `index.html#raid-1`) is copied to the clipboard.
3. Paste the URL into the browser address bar, refresh, and verify:
   - The page automatically navigates to "Raid Campaigns" (`#past-raids` section is active).
   - The specific raid card scrolls smoothly into view.
   - The card displays a brief glow animation (highlight).
