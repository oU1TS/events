Copy the theme toggle switcher, updated layout actions, and explicit theme styling overrides from the personal repository (`b1tranger.github.io`) to the events repository (`[oU1TS]\events`).

## Proposed Changes

### HTML & CSS

#### [MODIFY] [render.html](Documents/GitHub/[oU1TS]/events/render.html)
- Add the theme toggle button `#theme-toggle-btn` and `#theme-toggle-icon` in the `.header-right` actions block, matching the layout in the personal repository.

#### [MODIFY] [css/render.css](Documents/GitHub/[oU1TS]/events/css/render.css)
- Add explicit variables overrides for `body.dark-theme` (matching the default dark theme values in `:root`).
- Append the `github-markdown-css` theme overrides for `body.light-theme .markdown-body` and `body.dark-theme .markdown-body`. This forces the CDN-loaded media-query stylesheet to render colors aligned with the class toggles.
- Add a global override for `.markdown-body pre` and `.markdown-body pre code` to guarantee code text is always light-colored on its dark background.
