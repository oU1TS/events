# Implementation Plan - Update Render Files

We need to update `render.html` and `render.js` in the events repository (`oU1TS/events`) with new features from the personal repository (`b1tranger.github.io`).

## Key Differences & Preservation Requirements

1. **Dependency Loading**:
   - **Personal Repository**: Loads dependencies (Marked, DOMPurify, Highlight.js, KaTeX, Font Awesome) from a local `dependencies/` directory.
   - **Events Repository**: Must load all dependencies from CDNs, as there is no local `dependencies/` folder.
2. **Paths**:
   - **Personal**: Script is located at `js/render.js`.
   - **Events**: Script is located at `render.js` in the root.
3. **Footer**:
   - **Personal**: `b1tranger's stash`
   - **Events**: `oU1TS Community`
4. **Theme Sync**:
   - **Events**: `render.js` has a `syncTheme()` function that must be preserved to match the events website theme.
