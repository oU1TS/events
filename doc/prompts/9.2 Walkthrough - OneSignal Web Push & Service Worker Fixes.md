## How to Find & Configure your `ONESIGNAL_APP_ID`

`cff59aa0-39c1-4d35-9560-b23409f6d014` is a placeholder App ID. To make push notifications active on your real OneSignal project:

1. **Sign in to OneSignal**: Go to [OneSignal Dashboard](https://dashboard.onesignal.com/).
2. **Create / Open Your App**: Select your application (or click **New App/Website** -> select **Web Push**).
3. **Copy the App ID**:
   - Go to **Settings** -> **Keys & IDs** in the left sidebar.
   - Copy the **OneSignal App ID** (a 36-character UUID like `12345678-abcd-1234-abcd-1234567890ab`).
4. **Update `index.html`**:
   - In [index.html](file:///c:/Users/.../Documents/GitHub/%5BoU1TS%5D/events/index.html#L26), replace `"cff59aa0-39c1-4d35-9560-b23409f6d014"` with your actual **App ID**.
5. **Set GitHub Repository Secrets** (for GitHub Actions automated alerts):
   - In your GitHub repo: Go to **Settings** -> **Secrets and variables** -> **Actions**.
   - Add Secret `ONESIGNAL_APP_ID`: paste your App ID.
   - Add Secret `ONESIGNAL_REST_API_KEY`: paste your REST API Key (found in OneSignal Dashboard under **Keys & IDs**).
